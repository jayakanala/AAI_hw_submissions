# Submissions — Week 1

Submit your Week 1 deliverable here.

## How to Submit

1. Fork the repo
2. Create a folder: `submissions/your-name/`
3. Add your agent code inside it
4. Open a Pull Request

## What to Include

```
submissions/your-name/
├── agent.py          ← your agent code
└── README.md         ← what domain you picked + sample outputs
```

## Checklist

- [ ] Agent has 3 domain-specific tools
- [ ] Tools have docstrings
- [ ] Tested with at least 3 different queries
- [ ] README explains your domain and shows sample output
